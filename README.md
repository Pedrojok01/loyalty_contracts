## Description

Initial Smart Contracts draft associated with the Meed Loyalty project. Not to be used for production (yet!).

## Compile:

```shell
yarn compile
```

## Test:

```shell
yarn test
```

## Slither-analyzer:

```shell
yarn slither
```

## Coverage:

```shell
yarn cover
```

## Deploy:

```shell
yarn deploy
```
