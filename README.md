# Meed Rewards | From SuperUltra

Initial smart-contracts draft for Meed Rewards. Not to be used for production (yet!).

## Compile:

```shell
yarn compile
```

## Test:

```shell
yarn test
```

## Slither-analyzer:

```shell
yarn slither
```

## Coverage:

```shell
yarn cover
```

## Deploy:

```shell
yarn deploy
```
